Тебе дано содержимое файла
Заполни все поля инструмента describe_file, опираясь только на текст файла
Используй канонический kind (section, paragraph, list, list_item, table, table_header, table_row, code, function, class, config; при сомнении — logic_block) там, где это помогает понять структуру файла
Верни результат одним вызовом describe_file строго по схеме инструмента, без свободного текста
Если данных нет — ставь пустые значения (пустая строка, пустой массив, false, 0)
Ничего больше не делай и не отвечай текстом
