# Навигация по кодовой базе

## Структура проекта
- backend/ - Java Spring Boot приложение
- docs/ - Документация проекта  
- frontend/ - TypeScript/React приложение

## Основные компоненты
- chat.py - главный интерфейс Gradio
- tools.py - инструменты для LLM
- retriever.py - поиск через Elasticsearch
- utils.py - вспомогательные функции

## Конфигурация
- docker-compose.yml - настройка сервисов
- requirements.txt - Python зависимости
- images/ - Docker образы для сервисов

## Данные
- data/ - данные Elasticsearch и Neo4j
- repos/ - исходный код для анализа
- logs/ - логи работы системы
