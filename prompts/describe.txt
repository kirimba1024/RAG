Тебе дан блок текста
Заполни все выбранные поля инструмента describe, опираясь на предоставленный текст
Верни результат одним вызовом describe строго по схеме инструмента
Ничего больше не делай и не отвечай текстом

Описания полей:
- name: Имя одним словом, отражающее суть.
- title: Заголовок, кратко описывающий назначение.
- description: Одна фраза о содержимом.
- summary: Краткое содержание и цель.
- detailed: Подробное описание структуры и смысла.
- language: Основной язык/диалект (java, ts, yaml, md, sql…).
- purpose: Зачем нужен файл.
- file_type: Общий тип содержимого.
- tags: Главные ключевые теги, метки.
- key_points: Ключевые тезисы.
- http_endpoints: HTTP endpoints (method path).
- apis: API спецификации/идентификаторы.
- entities: Сущности/доменные объекты.
- imports: Импорты.
- dependencies: Зависимости.
- permissions_roles: Права/роли.
- config_keys: Ключи конфигов.
- feature_flags: Фичефлаги.
- todos: TODO/FIXME/XXX.
- edges: Функциональные зависимости (кто вызывает/зависит/использует), формат: typeA@nameA→typeB@nameB (напр. python-function@foo→sql-select@get_users).
- likely_queries: Ожидаемые вопросы по файлу.
- io: I/O артефакты (таблицы, топики, файлы, URLs).
- symbols: Символы/идентификаторы (универсально).
- security_flags: Флаги безопасности (pii, secrets, crypto, authz, audit).
- has_documentation: Есть ли заметная документация/комментарии.
- layer: Архитектурный слой (frontend/backend/data/ops и т.п.).
- bm25_boost_terms: Термы для буста BM25.
- domain_objects: Предметные сущности.
- complexity: Оценка сложности/LOC.
- confidence: Уверенность в корректности (0–1).
- vulnerabilities: Список уязвимостей (краткие описания).
- secrets_found: Найденные секреты.
- table_columns: Имена столбцов (если это таблица/SQL).
- improvements: Предложения по доработке.
- potential_bugs: Возможные баги.
- function_names: Имена функций/методов.
- class_names: Имена классов/интерфейсов/структур.
- variable_names: Имена переменных и констант.
- notes: Полезные заметки.
- conclusions: Выводы по файлу.
- open_questions: Вопросы и непонятности.
- highlights: Самое важное по файлу (коротко).
- anchors: Якоря для поиска.
