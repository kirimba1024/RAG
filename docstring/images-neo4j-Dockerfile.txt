НАЗВАНИЕ:
Neo4j Docker Image — граф база данных с APOC

SUMMARY:
Neo4j 5.15 с APOC плагином для расширенной функциональности графовых операций.

ВЫВОДЫ:
- Базовый образ: neo4j:5.15-community.
- APOC плагин: скачивается и устанавливается в /var/lib/neo4j/plugins/.
- Конфигурация: NEO4J_AUTH=neo4j/password, NEO4J_PLUGINS=["apoc"].
- Порт: 7474 (HTTP), 7687 (Bolt).
- USER neo4j для безопасности.

АНТИПАТТЕРНЫ:
- Использовать root пользователя в production.
- Пропускать установку APOC — ограничивает функциональность графовых запросов.

ЗАМЕТКИ:
- APOC версия: 5.15.0.0 совместимая с Neo4j 5.15.
- Плагин активируется через NEO4J_PLUGINS environment variable.
- Граф используется для хранения структурных связей между сущностями кода.
