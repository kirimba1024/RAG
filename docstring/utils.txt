НАЗВАНИЕ:
Utils — вспомогательные функции и константы

SUMMARY:
Общие утилиты и конфигурационные константы для RAG системы.
Содержит настройки подключения к базам данных, пути к файлам, вспомогательные функции.

ВЫВОДЫ:
- Константы подключения: ES_URL, NEO4J_BOLT_URL, NEO4J_USER, NEO4J_PASS.
- Пути: KNOWLEDGE_ROOT для доступа к файлам кодовой базы.
- Функции: to_posix для нормализации путей, load_prompt для загрузки промптов.
- Конфигурация: настройки для всех внешних сервисов в одном месте.
- Безопасность: централизованное управление учетными данными.

АНТИПАТТЕРНЫ:
- Хардкодить URL и пароли в коде — используем константы.
- Дублировать конфигурацию в разных модулях — централизованное управление.
- Не нормализовать пути — проблемы с кроссплатформенностью.

ЗАМЕТКИ:
- KNOWLEDGE_ROOT: Path объект для безопасной работы с путями.
- to_posix: конвертация Windows путей в Unix формат для Docker.
- load_prompt: кэширование промптов для избежания повторного чтения файлов.
- Конфигурация: все настройки в одном месте для легкого изменения.
