НАЗВАНИЕ:
Check Script — проверка состояния системы

SUMMARY:
Скрипт для проверки работоспособности всех компонентов RAG системы.
Проверяет подключение к Elasticsearch, Neo4j, доступность индексов и данных.

ВЫВОДЫ:
- Elasticsearch: проверка подключения, статуса кластера, наличия индексов.
- Neo4j: проверка подключения, выполнения тестовых запросов.
- Индексы: проверка количества документов в ES индексах.
- Граф: проверка количества узлов и связей в Neo4j.
- Результат: детальный отчет о состоянии всех компонентов.

АНТИПАТТЕРНЫ:
- Пропускать проверку критических компонентов — может скрыть проблемы.
- Не обрабатывать ошибки подключения — непонятные сообщения об ошибках.
- Проверять только один компонент — система работает как единое целое.

ЗАМЕТКИ:
- Таймауты: разумные лимиты времени для проверки подключений.
- Детализация: конкретные цифры (количество документов, узлов, связей).
- Цветной вывод: зеленый для успеха, красный для ошибок.
- Автоматизация: можно использовать в CI/CD для проверки деплоя.
