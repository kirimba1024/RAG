НАЗВАНИЕ:
Secret Scanner — сканер секретов в knowledge base

SUMMARY:
Утилита для аудита knowledge/ на наличие незамаскированных секретов.
Использует те же паттерны что и mask.py, но только для отчёта без модификации.

ВЫВОДЫ:
- Использует SECRET_PATTERNS из mask.py для консистентности проверки.
- Только чтение и отчёт — не модифицирует файлы.
- Rich progress bar для визуализации процесса сканирования.
- classify_secret_type: эмодзи-классификация для быстрой идентификации типа секрета.
- Выводит файл:строка + тип + превью находки (макс 120 символов).

АНТИПАТТЕРНЫ:
- Дублировать паттерны вместо импорта из mask.py — риск рассинхронизации.
- Модифицировать файлы в check утилите — нарушение принципа единственной ответственности.

ЗАМЕТКИ:
- Запускать после mask.py для проверки качества маскировки.
- Или перед mask.py для аудита перед очисткой.
- Игнорирует UnicodeDecodeError и OSError — только валидные текстовые файлы.
