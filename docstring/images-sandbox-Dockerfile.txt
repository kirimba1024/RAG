НАЗВАНИЕ:
Sandbox Docker Image — изолированная среда для команд

SUMMARY:
Alpine Linux с набором Unix утилит для безопасного выполнения команд от LLM.
Рабочая директория /app, knowledge/ монтируется read-only.

ВЫВОДЫ:
- Базовый образ: alpine:latest (минимальный размер).
- Утилиты: grep, findutils, coreutils, gawk, sed, bash, curl, wget, jq, tree, file, diffutils, less, vim.
- Рабочая директория: WORKDIR /app.
- Безопасность: read-only файловая система, user="nobody".
- Entrypoint: /usr/local/bin/entrypoint.sh для инициализации.

АНТИПАТТЕРНЫ:
- Устанавливать git — не используется в sandbox.
- Использовать полные образы (ubuntu/debian) — избыточный размер.
- Пропускать установку утилит — ограничивает функциональность команд.

ЗАМЕТКИ:
- Alpine пакеты: gawk вместо awk, findutils для find/xargs.
- Безопасность: контейнер запускается с read_only=True, network_mode="none".
- Производительность: быстрый старт благодаря Alpine, минимальный размер образа.
- Функциональность: полный набор Unix утилит для анализа кода и файлов.
