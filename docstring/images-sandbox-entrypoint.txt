НАЗВАНИЕ:
Sandbox Entrypoint — инициализация изолированной среды

SUMMARY:
Проверяет наличие knowledge данных в /app и выводит доступные команды.
Контейнер остается активным для выполнения команд через exec_run.

ВЫВОДЫ:
- Проверка данных: если /app пустой, выводит предупреждение.
- Read-only безопасность: не пытается изменять права доступа к файлам.
- Список команд: выводит доступные утилиты для LLM.
- Keep-alive: exec sleep infinity для поддержания контейнера активным.
- Безопасность: не создает файлы в read-only файловой системе.

АНТИПАТТЕРНЫ:
- Пытаться изменять права доступа к read-only файлам — вызывает ошибки.
- Создавать файлы в /app — read-only файловая система.
- Завершать контейнер после инициализации — нужен keep-alive для exec_run.

ЗАМЕТКИ:
- Knowledge монтируется через docker-compose volume: ./knowledge:/app:ro.
- Команды выполняются через container.exec_run() с timeout 30s.
- Пользователь: nobody для дополнительной безопасности.
- Функциональность: полный набор Unix утилит для анализа кода.
