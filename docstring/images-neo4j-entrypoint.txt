НАЗВАНИЕ:
Neo4j Entrypoint — инициализация графа знаний

SUMMARY:
Запускает Neo4j с предустановленными индексами и ограничениями для графа знаний.

ВЫВОДЫ:
- Ожидание готовности: retry loop 60 секунд через cypher-shell.
- Создание индексов: CREATE INDEX IF NOT EXISTS для entity_id, relationship_type.
- Создание ограничений: UNIQUE для entity_id, NOT NULL для обязательных полей.
- Инициализация данных: выполнение init.cypher скрипта.
- exec передает управление официальному docker-entrypoint.sh.

АНТИПАТТЕРНЫ:
- Запускать инициализацию синхронно — блокирует старт Neo4j.
- Не использовать exec — некорректная обработка SIGTERM при остановке контейнера.
- Пропускать создание индексов — медленные запросы к графу.

ЗАМЕТКИ:
- Retry loop: 60 попыток с интервалом 1 секунда ожидает готовность Neo4j.
- Индексы: ускоряют поиск по entity_id и relationship_type.
- Ограничения: обеспечивают целостность данных графа.
- init.cypher: содержит начальные данные или структуру графа.
